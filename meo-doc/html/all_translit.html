Транслитерация текста [all_translit]¶

Утилита для транслитерации любого текста в текстовых файлах.

Скачивание¶

Для скачивания программы нажмите [здесь]

Если скачивания не произошло или архив оказался битым, то скачать утилиту можно следующим способом:

Зайти в репозиторий [all_translit]

Вам потребуется два файла: all_translit.py и all_translit.bat

Скачать их можно кликнув по каждому из файлов левой кнопкой мыши, на октырвшемся файле нажать кнопку Raw и комбинацией ctrl+s сохранить файлы в какой-нибудь папке.

Подсказка

Скриншоты представлены для утилиты csv_encoder. Так как набор действий одинаков, скриншоты решено оставить.

Установка¶

Для работы приложения требуется установленный Python 2.7.x.

Если его нет, то заходим на [официальный сайт] . Скачиваем текущую версию Python 2.7.x

Предупреждение

Убедитесь, что скачиваете Python версии 2, а не 3

После загрузки запускаем установщик и следуем инструкции.

Подсказка

В установщике возможно нужно будет выбрать Выборочная установка или Custom, для того, чтобы указать путь установки.

Предупреждение

Важно, чтобы путь установки был в корень диска С, то есть путь должен выглядеть так: C:\Python27. Если вы не уверены, что сможете это сделать, позовите того, кто сможет.

Следующим шагом необходимо скачать модифицированный Python [здесь], распаковать его и слить с оригинальным.

Для этого нужно открыть файловый менеджер и зайти в корень диска C:\.

Подсказка

Зайти в Пуск -> Компьютер -> Локальный диск (C:)

Архив Python27.rar распаковываем на рабочий стол.

После чего перетащить распакованную папку Python27 в корень диска C:\ и на все предложения о замене файлов и слиянии папок нужно согласиться, предварительно поставив галочки типа сделать это для всех файлов и папок, чтобы компьютер сделал слияние сам.

Использование¶

Для запуска утилиты кликните дважды на файл all_translit.bat. Нажмите кнопку Выбрать файл, откроется диалоговое окно файлового менеджера, где предлагается выбрать файл, который нужно тренслитерировать.

Когда найдете файл и выделите его однократным нажатием левой кнопкой мыши нажмите кнопку Открыть. Как только в окне программы появится надпись Файл открыт успешно можно начинать транслитерирование.

Для этого нажмите кнопку Кодировать. Если все пройдет успешно, то откроется диалоговое окно файлового менеджера для сохранения уже исправленного файла. Необходимо указать место сохранения и его имя.

Ошибки¶

В случае появления ошибок рекомендуется обратится к разработчику. Если ошибка появится в диалоговом окне с надписью, что ее необходимо отправить разработчику, то текст ошибки можно просто отправить по электронной почте или в скайп. В случае же, если программа просто не открывается или ведет себя странно и ничего не делает, лучше обратится к разработчику.

Подсказка

Контактные данные разработчика находятся на главной странице. Для того, чтобы перейти туда, необходимо кликнуть на лого в верхнем левом углу.
