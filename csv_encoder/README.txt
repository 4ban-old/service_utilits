Установка:
Скачать python можно по ссылке https://www.python.org/ftp/python/2.7.10/python-2.7.10.msi
Для установки Python запустите python-2.7.10.msi, 
проследите, чтобы путь установки был C:\Python27
Следуйте инструкции установщика.

Для установки библиотек необходимо запустить автоматический установщик, кликнув дважды по файлу instal.bat,
либо сделать это вручную. Для этого необходимо открыть командную строку комбинацией клавиш win+r
и ввести в открывшемся окошке команду cmd.
Либо в меню Пуск:
Пуск -> Все программы -> Стандартные -> Командная строка

В открывшемся окошке ввести команду cd C:\Python27\Scripts и нажать Enter
Далее ввести поочередно команды и дождаться завершения каждой команды.:
pip install ttk
pip install Tkinter
pip install tkFileDialog


Для удобного запуска приложения, необходимо кликнуть правой кнопкой мыши на файл encoder.py -> открыть с помощью -> выбрать программу -> pythonw.exe

Если данной программы нет в списке, нужно кликнуть обзор, пройти в папку C:\Python27 (выбрать мой компьютер -> жесткий диск C -> Python27) и выбрать pythonw.exe.

Во время работы может появится окно с ошибкой так как не все символы можно перекодировать в cp1251 и разработчик не учел их все сразу.
Если во время работы появилось такое окно с ошибкой, информацию нужно скопировать и отправить разработчику. Новый символ будет добавлен в течение 5 минут.
