Перед запуском необходимо установить python интерпретатор 
и несколько библиотек. Python можно скачать по ссылке:
https://www.python.org/ftp/python/2.7.10/python-2.7.10.msi

Для установки Python запустите python-2.7.10.msi, 
проследите, чтобы путь установки был C:\Python27
Следуйте инструкции установщика.

Для установки библиотек необходимо запустить автоматический установщик, кликнув дважды по файлу instal.bat, либо сделать это вручную. Для этого необходимо открыть командную строку 
комбинацией клавиш win+r и ввести в открывшемся окошке команду cmd либо в меню Пуск:
Пуск -> Все программы -> Стандартные -> Командная строка

В открывшемся окошке ввести команду cd C:\Python27\Scripts и нажать Enter
Далее ввести поочередно команды и дождаться завершения каждой команды.:
pip install lxml
pip install requests
